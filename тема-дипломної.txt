Тема дипломної роботи:
Розроблення web-орієнтованої системи підтримки продажів малого бізнесу
з функціями обліку товарів, замовлень та клієнтської бази

Diploma topic:
Development of a web-based system for small business sales support
with functions for managing inventory, orders, and customer database

────────────────────────────────────────────────────────────
ОПИС ПРОЕКТУ
────────────────────────────────────────────────────────────

Мета роботи: розробити web-систему для автоматизації продажів малого бізнесу,
що забезпечує облік товарів, обробку замовлень та ведення клієнтської бази
з розмежуванням доступу між покупцями та адміністратором.

Об'єкт дослідження: процеси управління інтернет-магазином (каталог товарів,
кошик, оформлення замовлень, облік клієнтів).

Предмет дослідження: методи та засоби розроблення web-орієнтованих систем
підтримки продажів на основі сучасних технологій.

────────────────────────────────────────────────────────────
ФУНКЦІОНАЛЬНІ МОЖЛИВОСТІ
────────────────────────────────────────────────────────────

Для покупця:
  • Реєстрація та авторизація
  • Перегляд каталогу товарів з фільтрацією та пошуком
  • Додавання товарів до кошика
  • Оформлення та відстеження замовлень
  • Перегляд історії покупок

Для адміністратора:
  • Управління товарами (додавання, редагування, видалення)
  • Управління категоріями та цінами
  • Перегляд і обробка замовлень
  • Ведення бази клієнтів
  • Звітність по продажах

────────────────────────────────────────────────────────────
ТЕХНОЛОГІЧНИЙ СТЕК
────────────────────────────────────────────────────────────

Backend:
  • Python, FastAPI, SQLAlchemy
  • PostgreSQL 16

Frontend (клієнтська частина):
  • React 19, TypeScript, Vite

Admin-frontend (адмін-панель):
  • React 19, TypeScript, Vite

Інфраструктура:
  • Docker Compose
  • REST API
